Island¶

mod

‘biosim.island’ provides the user with the annual cycle on Rossumøya.

This script provide the annual cycle of events that accrue on Rossumøya ever year. The island of Rossumøya is also made in this script.

This file can be imported as a module and contains the following class:

  • Island - Class that contains the ways of making the map of Rossumøya and setting its

population. It also contains the function that makes the annual cycle of Rossumøya.

Notes

To run this script, its required to have ‘numpy’, ‘matplotlib.pyplot’ and ‘textwrap’ installed in the Python environment that your going to run this script in.
